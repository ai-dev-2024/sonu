# Installing Sonu Desktop v3.x.x

**Windows**: Run installer → SmartScreen warning → "More info" → "Run anyway"  

**macOS**: Open DMG → Drag to Applications → Right-click → "Open" → "Open"  

**Linux**: `chmod +x Sonu-Voice-Typing-*.AppImage && ./Sonu-Voice-Typing-*.AppImage`

**Why unsigned?** Sonu is free/open-source. Code signing costs $300/year. Your security comes from transparent code at https://github.com/ai-dev-2024/sonu

**Verify**: `sha256sum Sonu-Voice-Typing-*` and compare to SHA256SUMS.txt
