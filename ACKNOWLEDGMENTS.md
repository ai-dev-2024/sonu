## Core Technologies

### OpenAI Whisper

The foundation of SONU's transcription capabilities is built on OpenAI's Whisper model, a state-of-the-art speech recognition system that provides:

- **High Accuracy**: Industry-leading speech recognition accuracy
- **Multi-language Support**: Support for multiple languages
- **Open Source**: Released as open-source for the community

**Repository**: [openai/whisper](https://github.com/openai/whisper)  
**License**: MIT License  
**Website**: [OpenAI](https://openai.com)

### Faster-Whisper

SONU uses faster-whisper, an optimized implementation of Whisper that provides:

- **CPU Optimization**: Efficient CPU-based inference using CTranslate2
- **Performance**: Faster transcription with lower memory usage
- **Reliability**: Stable and well-maintained implementation
- **Model Management**: Automatic model downloading and caching

**Repository**: [guillaumekln/faster-whisper](https://github.com/guillaumekln/faster-whisper)  
**License**: MIT License  
**Author**: Guillaume Klein

### CTranslate2

The efficient inference engine powering faster-whisper:

- **CPU Optimization**: Highly optimized CPU inference
- **Performance**: Fast and memory-efficient
- **Open Source**: Free and open-source
- **C++ Backend**: High-performance C++ implementation

**Repository**: [OpenNMT/CTranslate2](https://github.com/OpenNMT/CTranslate2)  
**License**: MIT License  
**Organization**: OpenNMT

### Electron

The desktop application framework that makes SONU possible:

- **Cross-platform**: Desktop application framework
- **Web Technologies**: Leverages familiar web technologies
- **Native Integration**: System-level integration capabilities
- **Active Development**: Continuously maintained and improved
