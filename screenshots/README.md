# SONU Screenshots

This directory contains screenshots of the SONU application for the GitHub repository.

## Screenshot Guidelines

- **Format**: PNG (preferred) or JPG
- **Size**: Minimum 1280x720, preferred 1920x1080
- **Quality**: High quality, no compression artifacts
- **Naming**: Use descriptive names (e.g., `main-window-light.png`)

## Required Screenshots

- `main-window.png` - Main application window
- `settings-page.png` - Settings page
- `model-selector.png` - Model selector tab
- `dictation-active.png` - Active dictation with live transcription
- `tray-menu.png` - System tray context menu
- `theme-light.png` - Light theme
- `theme-dark.png` - Dark theme

## Taking Screenshots

See [scripts/take-screenshots.md](../scripts/take-screenshots.md) for detailed instructions.

## Adding Screenshots

1. Take screenshots using Windows Snipping Tool or Print Screen
2. Save them in this directory with descriptive names
3. Update `README.md` to include the screenshots
4. Commit and push to GitHub
