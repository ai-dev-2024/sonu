# Legacy Files Notice

**⚠️ IMPORTANT: These root-level files are LEGACY/BACKUP files**

## Do NOT Use These Files for Active Development

The root-level files (`main.js`, `index.html`, `renderer.js`, etc.) are **legacy/backup files** and should **NOT** be used for active development.

## Active Development Locations

### Desktop App (v3.x) - DEFAULT
- **Use files in**: `apps/desktop/`
- **Main file**: `apps/desktop/main.js`
- **UI file**: `apps/desktop/index.html`
- **Start command**: `cd apps/desktop && npm start`

### Mobile App (v4+) - Only When Requested
- **Use files in**: `apps/mobile/`
- **Only switch when explicitly told "work on mobile app"**

These root-level files are kept for:
- Historical reference

**They are NOT the active development files.**
